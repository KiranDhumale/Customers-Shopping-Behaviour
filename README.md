# Customers-Shopping-Behaviour
This project focuses on customer behavior analysis using Python, Pandas, SQL, and Power BI. 
The dataset was cleaned, transformed, and manipulated using the Pandas library to handle missing values, duplicates, and inconsistent data.
Exploratory data analysis was performed to identify key business insights based on gender, age group, purchasing patterns, and customer subscriptions.
The analysis also highlights which shipping modes are most preferred by customers and how different customer segments contribute to sales.
SQL queries were used to extract meaningful insights and support data-driven decision-making.
An interactive Power BI dashboard was created with filters, charts, and KPIs to visualize trends and compare customer behavior effectively.
The dashboard helps in understanding customer preferences, subscription impact, and overall business performance.
